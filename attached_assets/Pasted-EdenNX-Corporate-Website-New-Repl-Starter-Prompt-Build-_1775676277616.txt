EdenNX Corporate Website — New Repl Starter Prompt
Build the EdenNX corporate marketing website. This is the parent company site for EdenNX, the organization behind the EdenRadar product suite. The audience is sophisticated biotech and pharma buyers, life science investors, potential enterprise partners, and talent. The site should feel premium, credible, and minimal. Not a startup landing page. A company that knows what it's doing.

Tech stack: React + Vite, TypeScript, Tailwind CSS, React Router (or wouter), Framer Motion v10 for scroll reveals. Deploy target is Vercel via GitHub, so the output must be a standard Vite SPA with no SSR dependencies.

Brand rules (non-negotiable):

Primary color: emerald green. Use hsl(142 52% 36%) as the primary brand green. Lighter variant hsl(142 65% 48%) for glows and accents.
Background: near-white hsl(0 0% 98%) in light mode. Dark mode: hsl(222 47% 7%).
No blue anywhere in the design. Not in buttons, not in links, not in accents.
No em dashes (-- is fine in comments, never in UI copy).
Never use the phrase "AI-powered." Describe capabilities concretely instead.
Typography: clean, professional. Use Inter or a system font stack. Headings bold, body text at comfortable line-height.
Subtle radar-ring background decoration (concentric circles, low opacity green, no animation on the corporate site).
Footer: "2026 EdenNX. All rights reserved." with links to LinkedIn and product site (edenradar.com placeholder).
